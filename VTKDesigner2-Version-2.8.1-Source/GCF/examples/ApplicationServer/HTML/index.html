Introduction

GCFApplicationServer is a tool that comes bundled with GCF, that allows you to host one or more components on it for access from desktop and/or HTML web-application(s). In this HTML page you can find links to few sample web-applications written using GCFMessage JavaScript library for accessing sample web-services hosted on GCFApplicationServer.

Click on any of the following links to launch the corresponding web-application

  1. SimpleWebService: This web-application issues calls to the one and only web-service method exposed by the SimpleWebService component.
  2. Fractal: This web-application makes use of the FractalGenerator component's web-service methods to generate fractals based on one or more input parameters.
  3. Dicom: This web-application makes use of the DicomSupport component's web-services to make available medical images for browsing for you.

Note: The look and feel of these web-applications have been deliberately kept simple in order to make understanding of the web-applications easy.
