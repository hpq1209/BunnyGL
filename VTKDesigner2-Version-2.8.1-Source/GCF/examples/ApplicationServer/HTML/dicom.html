Web-Application for DICOMService component

Please ensure that GCFApplicationServer is running and that the DICOMService.Core and DICOMService.PatientList objects are listed available web-service objects in the application server.

If you dont find DICOMService.PatientList web-service object listed in the application server, then it means that VTK5 Library was not found during the compilation of GCF. Please set the VTK5_SDK_ROOT environment variable to the directory where both Debug and Release versions of the VTK 5.4 SDK can be found.

If you are able to see DICOMService.PatientList listed as a web-service object; then click on the link below to launch the DICOM Web Application
